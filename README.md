# Sovereign Watch

Real-time US Treasury debt analytics dashboard.

## Features

- Debt Composition Sankey Diagram
- 50-Year Historical View
- Maturity Wall Analysis
- Auction Demand Tracking
- AI Analyst (Gemini)

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript / Tailwind CSS
- Plotly.js / Shadcn UI
- Vercel AI SDK

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000


```

## Data Sources

- Treasury Fiscal Data API (live)
- Fed Z.1 / TIC (static estimates)
